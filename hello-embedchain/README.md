# hello-embedchain
[[Demo](https://bistec-chat.streamlit.app/)]

## Create environment
```shell
python -m venv .venv
.venv/Scripts/Activate.ps1  
```

## Install Dependencies
```shell
pip install -r requirements.txt
```

## Install OpeAi Version
```shell
pip install "openai<1.0.0"
```

## Run app
```bash
streamlit run app.py
```
